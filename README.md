# SNAKE GAME

Implementação do Jogo SNAKE utilizando Python 3 e PyGame 2.1.2.

### COMO JOGAR

* Execute o script snake.py através da seguinte linha de comando:

```
$ python snake.py
```

* Controle o movimento da cobra utilizando as setas do teclado.
